B1数学復習

Chef : Yohta

1. 概要

微分積分・線形代数は幅広い分野の基礎となる数学である。しかし、自分の興味関心に直接的に関係のない微分積分・線形代数の諸定理は重要であっても忘れてしまうことがあるだろう。本ゼミでは、これらの復習を「他者の興味関心を借りる」事で行う。 まず、シラバスを基に微分積分・線形代数の基本と全体像を思い出す。次に各々が興味関心があるテーマの「微分積分・線形代数との関わり」を発表する。次回以降は初回に出たテーマに対してくじ引きで発表の担当を決める。つまり、他者の興味関心のあるテーマに関して調べて、発表を行うわけだ。これを繰り返すことにより、普段の自分とは異なる視点から微分積分・線形代数を活用する方法を理解する。調査と発表を具体的なテーマに対して行うことを通し、学部一年生の数学をより深く身につけることを目的とする。
